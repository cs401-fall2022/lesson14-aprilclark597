Types

TypeScript makes developers lives easier by enforcing strict type. JavaScript provides language primitives like string and number, but it doesn’t check that you’ve consistently assigned them. TypeScript does this for us! This means that an existing JavaScript code is not wasted - it is also TypeScript code. The main benefit of TypeScript is that it can highlight unexpected behavior in your code, lowering the chance of bugs.
